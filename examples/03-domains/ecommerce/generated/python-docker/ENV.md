## Shared Infrastructure

The following environment variables are set automatically in `docker-compose.yml`:

| Variable | Description |
|----------|-------------|
| `{BLOCK}_DATABASE_URL` | Per-block database connection string (e.g. DB_DATABASE_URL) |
| `PORT` | Application HTTP port |
| `SERVICE_NAME` | Service identifier |
| `REDIS_URL` | Redis connection URL |
| `KAFKA_BROKERS` | Kafka broker addresses |
| `BROKER_URL` | Broker connection URL (RabbitMQ) |
| `NOSQL_URI` | NoSQL database connection URI |
| `NOSQL_DATABASE` | NoSQL database name |
| `INTERNAL_API_TOKEN` | Token for internal API calls (X-Internal-Token); required by services with internal endpoints and callers |
